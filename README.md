# Llama-Pan-LATAM

# Reproduzindo

```shell
$ pip install uv
$ uv venv --python 3.11.5
$ source .venv/bin/activate 
```

Caso você use uma distro linux, você deve instalar:

`sudo apt-get install libportaudio2`
